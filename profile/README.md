## 先搞清楚一件事：VPS China 的核心问题是"线路"

在说套餐之前，有必要先聊聊为什么在中国用海外 VPS 总是"看运气"。

中国大陆与境外的网络连接，走的是国际出口带宽。高峰期（晚上 7 点到 11 点）拥堵是常态，普通线路的丢包率能飙到 30% 以上。这个丢包率意味着什么？简单说：SSH 连接频繁断开、网页加载要转好几圈、视频会议卡成幻灯片。

搬瓦工的核心卖点就是 CN2 GIA（全球互联接入）线路。中国电信专门为高质量商业需求铺的专用通道，绕开了拥堵的公共骨干网，延迟低、丢包少、稳。这不是营销说辞——跑 MTR 路由追踪可以实测。

搬瓦工目前的套餐体系，本质上就是**按线路质量分层定价**：

- **KVM 基础款**：普通国际线路，适合对速度要求不高的场景
- **CN2 GT 套餐**：电信 CN2 GT 中端线路，比普通线路好，性价比均衡
- **CN2 GIA-E 套餐**：三网 CN2 GIA 高端线路，公认最具性价比的主力选择
- **香港/东京 Ultra 套餐**：延迟最低，适合对速度要求极苛刻的场景
